Pour Extension Setup Chrome

Pour is a Chrome extension that takes forecast files generated by Predict and automatically imports
them into OnePortal - accurately, consistently, and in seconds.

  1. Log into Predict & Pour (Home page) so the extension unlocks.
  2. Install the Pour Chrome Extension (one-time).
  3. Open the extension popup and upload your forecast file.
  4. Go to OnePortal Forecasting page and click POUR.
Install Pour Extension Open OnePortal
Tip: OnePortal forecasting page is typically https://oneportal.beertech.com/forecasting/forecast
